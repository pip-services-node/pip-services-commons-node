Class RandomDateTime

Provides methods that can be used for generating random Dates/DateTimes, as well as updating existing DateTimes by generating values in the range of 'original DateTime' ±'delta/range'

Static updateDateTime

  • updateDateTime(value: Date, range?: number): Date
    • Defined in random/RandomDateTime.ts:64

    Generates a new DateTime that will differ from 'value' by a maximum of ±'range'. If 'range' is omitted (or 0), then the generated DateTime will differ from 'value' by ±10 days.

    Parameters

    • value: Date

      DateTime to update.

    • Default value range: number = null

      (optional) defines the maximum amount of days by which the new DateTime can differ from 'value'. If range is a negative number, 'value' will be returned. Defaults to 10 days if omitted or zero.

    Returns Date

    updated DateTime.
