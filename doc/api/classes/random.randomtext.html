Class RandomText

Provides methods that can be used for generating random strings, which can contain: colors, names (first + last, with/without prefixes/suffixes), objects, adjectives, verbs, phrases (words separated by spaces), phone numbers, email addresses, sequences of words (CamelCase), or texts.
